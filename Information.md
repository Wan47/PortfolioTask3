# Task 3: Deploy an App to Render Cloud Platform

## Task Grading Requirement

To achieve a specific grade for this task, students must complete all lower-level sub-tasks leading up to that level. The final result for the portfolio is the sum of the grades for each task.

### Task 3.1 – Pass
* **Account Creation:** Create a Render account at [https://render.com](https://render.com).
* **Git Integration:** Connect a Git repository (GitHub, GitLab, or Bitbucket) to Render.
* **Initial Deployment:** Deploy any existing web app (e.g., a static site or sample app) to Render.
* **Verification:** Verify the deployment via the public Render URL.

### Task 3.2 – Credit
*Complete Task 3.1 and:*
* **App Development:** Develop a simple web app (e.g., Python/Flask, Node.js, or PHP).
* **Configuration:** Configure environment variables and build/start commands in Render.
* **Automation:** Deploy your app to Render via Git push (auto-deploy).
* **Service Management:** Suspend or delete the service to demonstrate deactivation.

### Task 3.3 – High Distinction
*Complete Task 3.2 and:*
* **Database Integration:** Add a persistent database (e.g., Render PostgreSQL or an external managed DB).
* **Data-Driven App:** Develop a web app that reads from and writes to the database.
* **Full-Stack Deployment:** Deploy the complete app (web service + database) to Render.
* **Live Demonstration:** Demonstrate the live app via its public Render URL with fully working database functionality.

*Note: There is no specific Distinction sub-task for this task.*
